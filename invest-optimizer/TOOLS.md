# Data Tools

Quantitative work that supports — but does not replace — the judgment calls in SKILL.md.

| Tool | What it does | When to reach for it |
|---|---|---|
| `qlib` (microsoft/qlib) | Factor/return backtests | Sanity-checking valuation or macro verdicts with historical factor data |
| `timesfm` (google-research/timesfm) | Time-series forecasting | Forecasting the metrics defined in METRICS.md to validate regime reads |
| `TradingAgents` (TauricResearch) | Multi-agent market narrative | Cross-reading agent-flow signal against the microstructure axis |
| `polymarket-cli` | Live prediction-market prices | Recession probability, rate-hike odds, sector-outcome markets (see Phase 2E) |

If a tool is unavailable, fall back to the manual metric checks in METRICS.md and state the gap. Never block a posture brief on a missing data source.
